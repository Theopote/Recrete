## 一句话

**Recrete（砼憶）** 是面向既有建筑更新的 AI 设计助手——读资料、理解建筑、诊断问题、生成多档改造方案，并支持团队轻量评审。

## 适合谁

- 做**城市更新 / 旧改 / 文保活化**的建筑师与 PM
- 手头有**不完整图纸**和**现场照片**，需要快速形成方向性方案
- 希望用 AI **缩短前期研判时间**，而不是替代最终设计决策

## 30 分钟能做什么

1. **一句话建项目** — 描述建筑背景，AI 自动整理 Building Memory
2. **上传旧图纸 / PDF / 照片** — 异步解读，提取关键信息
3. **生成三档改造方案** — 轻量 / 中等 / 深度，带 AI 推荐理由与对比
4. **方案下留言评审** — @ 同事、关联风险点

## 与通用 AI 聊天有何不同

| | 通用 ChatGPT | Recrete |
|---|-------------|---------|
| 项目上下文 | 每次重讲 | 持久 Building Memory |
| 资料 | 粘贴片段 | 上传图纸/文档管线 |
| 输出 | 自由文本 | 结构化方案 + 诊断 + 任务 |
| 协作 | 无 | 项目内评审留言 |
| 数据 | 在第三方 | 按事务所隔离存储 |
